问题 A: Complete binary tree [Easy I]

问题 A: Complete binary tree [Easy I]

时间限制: 1 Sec  内存限制: 128 MB
提交: 0  解决: 0
[提交][状态][讨论版]

题目描述

Lanran has a binary tree with nn nodes, she does not know whether this tree is a complete binary tree or not. She turns to you for help. We guarantee that the input is a binary tree.

输入

The first line will be an integer TT, which is the number of the test cases.(1≤T≤14)(1≤T≤14) For each test case, the first line will be an integer nn.(1≤n≤150000)(1≤n≤150000)Then followed by nn lines, each line will be two integers xx and yy, the i-th line means the left child of node i is xx, the right child of node i is yy. If node i has no left child, then xx will be 0. If node i has no right child, then yy will be 0.

输出

For each test output Yes or No in one line.

样例输入

1
5
2 3 
4 0
5 0
0 0
0 0

样例输出

No
